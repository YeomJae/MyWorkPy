### 4. 기존 print 함수를 debug_print()로 교체하기
 - print("프린트할 내용") ==> debug_print("프린트할 내용", False)
      - True : console에도 나오고 log_yyyymmdd.txt에도 저장
      - False : console에는 나오지 않고 log_yyyymmdd.txt에도 저장
